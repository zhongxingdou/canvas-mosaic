Canvas-mosaic
=============

Getting started
-----------
```html
<script src='mosaic.js'></script>

<div id="myCanvasWrap">
</div>

<script>
new Mosaic(myCanvasWrap, 'mm2.jpeg', {
                drawInterval: 50
			})
</script>
```

Run example
-----------

example must run as a webserver page
